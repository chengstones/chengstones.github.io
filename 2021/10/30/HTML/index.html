HTML

发表于 2021-10-30 更新于 2021-11-04 分类于 CS

URL：https://developer.mozilla.org/zh-CN/docs/Web/HTML

摘要：HTML (HyperText Markup Language) 不是一门编程语言，而是一种用来告知浏览器如何组织页面的标记语言。HTML 可复杂、可简单，一切取决于开发者。它由一系列的元素（elements）组成，这些元素可以用来包围不同部分的内容，使其以某种方式呈现或者工作。 一对标签（ tags）可以为一段文字或者一张图片添加超链接，将文字设置为斜体，改变字号，等等。

学习小tip：用记事本编辑 HTML 语言，用 chrome 打开，可以实时查看运行结果。

几种前端语言

  • HTML 网页含义和结构
  • CSS 描述网页的表现与展示
  • JavaScript 功能和行为

Section 4 建立超链接

  1. 链接的解析
    • 属性 href：指向希望链接的网址

  2. title 属性 添加支持信息
    • 效果：当鼠标悬停时，将作为提示信息出现
  3. 块级链接
    • 将整块元素作为 < a > 的 string 属性，可将整块元素都作为标签

  4. 下载链接 download 属性

    • 缺省时默认为预览状态
    • 否则强制下载链接，并命名为属性的值

  5. 电子邮件链接

    • < a > 和 mailto 方案
    • 使用形式为 mailto: link 格式 ， link 是缺省的

Section 6 文档与网站架构

  1. 文档的基本组成部分
    • 页眉 < header >

    • 页脚 < footer >
  2. HTML 布局元素细节
    • < main > 只能用一次，且位于 < body > 中
    • < article > 仅包围一篇文章
    • < section > 组织结构使其按功能划分
    • < aside > 包含简介信息
    • < nav > 包含页面主导航，不能包含二级链接等内容
  3. 无语义元素

    • < hr >
