查找

  • 私有函数，仿前序遍历
  • 参数 x-> 寻找值;
  • 参数 node* t -> 需要开始查找的子树根结点

    清空，析构，剪枝

  • 仿前序遍历：剪枝左子树，剪枝右子树，析构自己

前序遍历

  • 先访问根结点，再访问左右子树
  • 具体实现 -> 递归 -> 传入参数结点，实现对应功能
  • 接口函数 包装 参数 void

层次遍历

  • 队列
  • 规则：出列头结点，访问该节点，将左右子节点入队
  • 起始条件：入队根结点；终止条件：队列空

建树

  • 面向控制台，队列
  • 参数 截止参数 flag
  • 规则：出列头结点，向控制台申请左右子节点，若非截止结点，则生成节点并入队
  • 起始条件：入队根结点；终止条件：队列空

非递归前序

  • 栈；参数 void
  • 规则：出栈顶，访问该节点，入栈右左节点
  • 起始条件：入队根结点；终止条件：队列空
